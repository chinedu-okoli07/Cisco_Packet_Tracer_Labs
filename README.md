# Cisco Packet Tracer Labs

This repository contains networking labs and simulations built using Cisco Packet Tracer.

---

## 🔧 Labs Included

### 🖥️ Basic LAN Setup
- Configured PCs, laptops, switches, and servers
- Assigned IP addresses
- Tested connectivity using ping commands

### 🌐 Multi-Network Topology
- Connected multiple networks using a router
- Configured routing and communication between devices
- Verified successful packet transmission

### 📡 Packet Simulation
- Used simulation mode to analyze network traffic
- Observed ICMP packet flow and connectivity events

---

- Cisco Packet Tracer
